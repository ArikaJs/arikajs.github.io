Routing

The most basic ArikaJS routes accept a URI and a closure, providing a very simple and expressive method of defining routes and behavior.

Basic Routing

The default ArikaJS routes are defined in your routes/web.ts and routes/api.ts files. The web.ts file defines routes that are for your web interface.

routes/web.ts
import { Route } from 'arikajs';

Route.get('/greeting', () => {
    return 'Hello World';
});

Available Router Methods

The router allows you to register routes that respond to any HTTP verb:

Route.get(uri, callback);
Route.post(uri, callback);
Route.put(uri, callback);
Route.patch(uri, callback);
Route.delete(uri, callback);

Route Parameters

Sometimes you will need to capture segments of the URI within your route. For example, you may need to capture a user's ID from the URL:

Route.get('/user/:id', (req) => {
    return 'User ' + req.params.id;
});
