Installation

ArikaJS is a web application framework with expressive, elegant syntax. We’ve already laid the foundation — freeing you to create without sweating the small things.

Meet ArikaJS

ArikaJS is a Node.js framework designed for extreme performance and developer happiness. It is built from the ground up to be type-safe, modular, and easy to use, drawing inspiration from industry leaders like Laravel and NestJS.

Prerequisites

Before creating your first ArikaJS project, you should ensure that your local machine has Node.js and npm installed. We recommend the latest LTS version of Node.js.

  • Node.js 20.x or higher
  • npm 10.x, Yarn, or pnpm

Your First ArikaJS Project

You may create a new ArikaJS project via the ArikaJS CLI. The CLI is available via npx, so there is no need to install it globally:

npx @arikajs/cli new my-app

After the project has been created, you should enter the project's directory and start the ArikaJS development server:

cd my-app
arika serve --dev

Initial Configuration

The ArikaJS framework requires very little configuration out of the box. However, you should check your .env file to ensure the APP_KEY is generated. You can generate a new key using the CLI:

arika key:generate
