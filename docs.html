Installation

ArikaJS is a high-performance web application framework for Node.js, engineered for stability, type-safety, and developer productivity.

Why ArikaJS?

ArikaJS bridges the gap between the simplicity of Express and the structured power of Laravel. Built entirely in TypeScript, it provides a driver-based architecture for Database, Cache, and Session management, ensuring your application can scale from a prototype to an enterprise system without refactoring.

System Requirements

Before deploying or developing with ArikaJS, verify that your environment meets the following baseline requirements:

  • Node.js: v20.11.0 or higher.
  • Package Manager: npm v10+, Yarn v3+, or pnpm v8+.
  • Operating System: Linux, macOS, or Windows (WSL2 recommended).

Scaffolding Your Application

The ArikaJS CLI is the primary tool for managing your application lifecycle. You can initialize a new project using the @arikajs/cli via npx:

Terminal
npx @arikajs/cli new my-awesome-app

This command prepares a fully structured monorepo or standalone application with all essential dependencies, TypeScript configuration, and a sample routing engine.

The Development Server

Once the project is created, enter the directory and launch the development server. ArikaJS includes a sophisticated watcher that handles hot-reloading for both your TypeScript logic and Ark Templates:

Terminal
cd my-awesome-app
npm run dev

Your application will be available at http://localhost:3000 by default.
