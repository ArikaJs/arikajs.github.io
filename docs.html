Installation

ArikaJS is a high-performance web application framework for Node.js, engineered for stability, type-safety, and developer productivity. It combines the expressive power of Laravel with the massive scalability of the Node ecosystem.

Our Philosophy

Modern Node.js development often feels fragmented. ArikaJS was built to provide a Batteries-Included experience. We believe that developers shouldn't have to piece together their routing, validation, and database layers for every project. By providing a structured, driver-based architecture, ArikaJS allows you to focus solely on your business logic.

Prerequisites

ArikaJS leverages the latest features of the Node.js runtime. Ensure your environment meets these requirements:

  • Node.js: v20.11.0 (LTS Iron) or newer.
  • TypeScript: v5.0 or newer (included in starter kits).
  • Package Manager: npm, Yarn (Berry), or pnpm.

Creating Your First Project

The easiest way to start is via our CLI tool. Use npx to initiate the scaffolding process without a global installation:

Terminal
npx @arikajs/cli new my-app

During the installation, the CLI will:

  1. Create your application directory.
  2. Scaffold the standard MVC directory structure.
  3. Initialize your package.json with core framework dependencies.
  4. Configure the tsconfig.json for optimal ArikaJS performance.
  5. Create a default .env file with a freshly generated application key.

The Development Workflow

ArikaJS includes a built-in development server powered by arikajs serve. This server monitors your app and routes directories for changes and performs hot-restarts automatically:

Terminal
cd my-app
npm run dev

By default, your application will be served at http://localhost:3000. You can change this port in your .env file.
