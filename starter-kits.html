Starter Kits

Start building instantly with the ArikaJS Application Skeleton. Our starter kits provide a batteries-included environment with authentication and database logic pre-configured.

The Application Skeleton

A fresh install of ArikaJS provides a modular TypeScript environment. It isn't just a folder structure; it's a pre-configured MVC application with support for:

  • Environment-specific configuration using .env.
  • Production-ready logging via the logger package.
  • A centralized Service Provider architecture.
  • Type-safe Routing and HTTP request handling.

CLI Scaffolding

Speed up your development using the Arika generator. You can generate essential framework components with a single command:

Generators
# Create a new controller
arika make:controller UserController

# Create a model and its migration
arika make:model Flight --migration

# Create a middleware
arika make:middleware VerifyToken

Database Ready

The standard starter kit includes a database/migrations folder pre-populated with basic User and Session tables to get you started with authentication immediately.
