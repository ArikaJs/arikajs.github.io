Frontend & Ark Templates

Build dynamic UIs with the powerful Ark engine. Designed for speed and developer convenience, Ark templates are fully compiled and cached for production.

Rendering Data

Data passed from your controllers can be rendered using the "Mustache" syntax. ArikaJS automatically escapes all output to prevent XSS attacks:

<h1>Hello, {{ $user->name }}</h1>

Directives

Ark provides a set of directives that allow you to add logic to your templates without complex JavaScript blocks.

Conditionals

@if($user->isAdmin)
    <div>Welcome, Administrator</div>
@else
    <div>Welcome, User</div>
@endif

Loops

<ul>
@foreach($users as $user)
    <li>{{ $user->name }}</li>
@endforeach
</ul>

Layouts & Sections

Maintain consistent headers and footers using the @layout directive. Sub-views can then "inject" content into specific sections:

profile.ark.html
@layout('layouts.app')

@section('content')
    <h1>User Profile</h1>
@endsection
