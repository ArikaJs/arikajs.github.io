Frontend

ArikaJS provides a robust set of tools for building your application's frontend, including a powerful template engine and asset bundling support.

Ark Templates

Ark is the powerful templating engine provided by ArikaJS. Ark templates are stored in the resources/views directory and use the .ark.html extension.

resources/views/welcome.ark.html
<!DOCTYPE html>
<html>
<body>
    <h1>Hello, {{ $name }}</h1>
</body>
</html>

Asset Bundling

ArikaJS includes built-in support for bundling your CSS and JavaScript assets. You can define your entry points in the arikajs.config.ts file and use the @asset directive in your templates:

<head>
    <!-- Load compiled assets -->
    {{ @asset('css/app.css') }}
</head>
